## QT串口波形调试上位机

可以直接通过串口打印向上位机发送数据（不需要使用任何帧头，帧尾），现阶段仅能输出打印一个数据



### 使用方法

**①点击Start USART，打开串口**

**②当显示USART OPEN SUCCESSFUL时，点击Open USART即可显示波形**
